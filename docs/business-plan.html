Section 1

Executive Summary

💡 The Opportunity: South Africa's construction industry loses 15-30% of project value to payment disputes. Itesho solves this by ensuring only compliant, Project Manager-approved work gets invoiced.

1.1 The Problem

  • Over 100,000+ construction companies in South Africa still use Excel and WhatsApp to manage BOQs
  • 15-30% of project value is lost to payment disputes
  • No compliance verification before work is submitted
  • No audit trail for approvals
  • Manual invoice calculations lead to errors and disputes

1.2 The Solution

Itesho is a cloud-based project control platform that:

  • Enforces compliance via a gatekeeper that blocks non-compliant users
  • Requires Project Manager approval before work can be invoiced
  • Auto-calculates invoices from approved quantities × rate
  • Provides full audit trails of every approval

1.3 Traction & Validation

  • Live deployment on Vercel (frontend) and Render (backend)
  • 7+ core modules fully functional (Dashboard, Projects, BOQ, RFQ, Gatekeeper, Daily Logs, Invoices)
  • B-BBEE Level 1 · 100% Black-Owned · CIDB registered
  • Founder credentials: 15+ years admin/records management, Construction Management Certificate (UCT), Software Development, Cloud Computing, Cybersecurity, Data Analytics certifications

Section 4

Products & Services

4.1 Core Modules (Live)

  • Dashboard: System status, project overview, progress tracking
  • Projects: Create and manage projects with BOQ hierarchy
  • Compliance Gatekeeper: Safety permits, insurance, licences, inductions
  • RFQ Management: Create, track, and award tenders
  • Bill of Quantities (BOQ): 3-level hierarchy (Section → Subsection → Line Items)
  • Daily Logs: Site capture with photos, weather, manpower, equipment
  • Submissions: PM approval workflow with audit trail
  • Invoices: Auto-generated from approved quantities

4.2 Technology Stack

  • Frontend: React + Vite (Vercel)
  • Backend: Django REST Framework (Render)
  • Database: PostgreSQL
  • Auth: JWT (djangorestframework-simplejwt)

4.3 Desktop Version (Roadmap)

  • Electron wrapper for Windows, macOS, Linux
  • Offline mode with automatic sync
  • Data sovereignty for clients with compliance requirements
