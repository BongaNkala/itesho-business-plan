Section 4

Products & Services

4.1 Platform Overview & Positioning

Itesho is a cloud-based project control platform designed for any project-based work — construction, public works, infrastructure, engineering, facilities management, and government projects.

🏛️ Not Just Construction: While Itesho's design was inspired by construction workflows, the platform applies to any sector that uses:
  • Bill of Quantities (BOQ) — For tracking work items, quantities, and costs
  • RFQs — For procurement and supplier management
  • Compliance — For safety, insurance, permits, and certifications
  • Daily Logs — For progress tracking and site/workspace reporting
  • Project Manager Approval — For certifying work before invoicing
  • Invoicing — For payment processing

4.2 Core Modules (Live as of June 2025)

📊 Module 1: Dashboard — Single-screen visibility into all projects and approvals

  • System status: Performance metrics, uptime, active projects
  • Portfolio overview: Total projects, overall progress, total budget, pending approvals
  • Project cards: Quick view of each project's status, budget, spend, pending items
  • Alerts: Compliance expirations, overdue approvals, budget warnings
  • Quick actions: Create new project, generate invoice, approve daily log

Business Impact: Project Manager can manage portfolio from anywhere.

📁 Module 2: Projects — Create and manage individual projects with BOQ hierarchy

  • Create new project with budget, timeline, location
  • Assign contractors, sub-contractors, project team
  • Manage project details: scope, start/end dates, stakeholders
  • View all projects in a list with status indicators
  • Archive completed projects

Business Impact: Single source of truth for project setup.

🔒 Module 3: Compliance Gatekeeper — Enforce compliance before work can be submitted

  • Compliance checklist per project: Safety file, insurance certificates, permits, inductions, licenses
  • Document upload with version control
  • Expiry date tracking with automatic renewal alerts
  • Access control: Blocks access to daily logs if compliance incomplete
  • Approval workflow: FM or Project Manager must approve each compliance item
  • Audit trail: Who approved what, when

Compliance Items Tracked: Safety certifications, Insurance (Liability, Workman's Comp, Professional Indemnity), CIDB grading proof, Worker inductions, Environmental permits, Statutory licenses

Business Impact: Compliance is verified before work starts.

📦 Module 4: RFQ Management — Create, distribute, and evaluate supplier quotes

  • Create RFQ with specifications and requirements
  • Distribute to multiple suppliers
  • Track responses and deadlines
  • Compare quotations side-by-side
  • Award contracts to selected supplier
  • Link awarded RFQ to BOQ line items

Business Impact: Transparent procurement. Faster quote turnaround.

📐 Module 5: Bill of Quantities (BOQ) — Track all work items, quantities, rates, and costs

  • Structure: Level 1 Sections → Level 2 Subsections → Level 3 Line Items
  • Fields: Item ID, Description, Unit, Planned Quantity, Rate, Approved Quantity, Total Cost, Status
  • Features: Expand/collapse sections, real-time progress tracking, budget tracking, export to Excel/PDF

Business Impact: Single source of truth for quantities.

📝 Module 6: Daily Logs — Site/workspace teams capture daily progress, resources, and photos

  • Select project and log date
  • Tabs: Work, Manpower, Equipment, Weather, Safety, Quality, Delays, Photos
  • Save draft, submit for approval
  • Timestamp on submission, contractor name auto-captured

Business Impact: Real-time visibility with photo evidence.

✅ Module 7: Submissions (Approval Workflow) — Project Manager reviews and approves daily logs before invoicing

  • Table of all submitted daily logs: Date, Contractor, Work Description, Quantities, Status
  • Filter by status: Pending, Approved, Rejected
  • One-click approval or rejection with comment field
  • Statuses: 📋 Pending → ✅ Approved → ❌ Rejected

Business Impact: Project Manager approves once, dispute-free.

💰 Module 8: Invoices — Auto-generate invoices from Project Manager-approved quantities

  • Auto-generated from Project Manager-approved daily logs
  • Invoice line items: BOQ item, approved quantity, rate, total
  • Status: Draft, Submitted, Paid, Overdue
  • Export to PDF, payment tracking

Calculation Logic:

Invoice Amount = Sum of (Approved Quantity × Rate) for all approved items in billing period

Business Impact: Invoices ready in 5 minutes. No disputes.

📊 Module 9: Reports — Monthly reporting for accountability and SACAP compliance

The Reports module auto-generates comprehensive monthly reports that provide full visibility into project performance, submissions, compliance, and finances.

Section 5

The Approval Chain

▐ ITESHO APPROVAL CHAIN ▐
STEP 1: COMPLIANCE GATEKEEPER
Project starts → Compliance checklist required
↓
Is compliance complete? → NO → Block access to daily logs
→ YES → Proceed to Step 2
 
STEP 2: DAILY LOG SUBMISSION
Site team submits daily log
- Work items completed
- Quantities recorded
- Photos attached
- Weather, manpower, delays noted
↓
Daily log submitted for Project Manager review
 
STEP 3: PROJECT MANAGER APPROVAL
Project Manager reviews daily log
- Compares quantities to site conditions
- Checks photos and evidence
- Verifies against BOQ
↓
Is everything accurate? → NO → Reject. Contractor resubmits.
→ YES → Approve. Quantities lock in.
 
STEP 4: AUTO-GENERATED INVOICE
Approved quantities + Rate = Invoice
Invoice = (Approved Qty × Rate) for all approved items
↓
Invoice ready for payment
│
↓
✅ PAYMENT (No disputes)
 
🔒 THE GOLDEN RULE:
No Compliance → No Daily Log → No Project Manager Approval → No Invoice
